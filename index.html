Project Abstract

In this project we are proposing a survey of neighboring sovereign nations to modern day Russia to test for any possible collateral Censorship. Given the socioeconomic state of some of these countries there may be limited infrastructure for Internet access requires their traffic to be routed through Russian territory. Russia actively censors various websites and Internet sources based on various reasons. We intend to use this to fulfill the experiment portion of project option 1 in the class forking the IClab software to complete our experiment. As part of option 1 we will also create a Web interface to run our experiment on IClab and a website presenting our final results.
